Eleventy
i use eleventy to generate this site. i followed this tutorial by whiona to learn how to set eleventy up but i use the following tutorial to update my site instead of Neocities CLI. special thanks to the definition lists markdown-it plugin for making writing this site in markdown possible and making updates 100x easier.
Deploy to Neocities
tutorial on how to use github to update your neocities website. i highly recommend following this tutorial and coding your website in VSCodium (basically visualstudio code without the microsoft telemetry). i found this page to be very helpful in getting github connected to vscodium when i was having issues. also install the live preview extension so you can live preview your code right as you’re coding. special thanks to the markdown all in one extension for live rendering of the definition list syntax.
Repth Themes
this website originally used theme 12 by repth. it has since been completely rewritten from scratch to be responsive, mobile friendly, use semantic tags, and gone through some design changes, but i’m leaving this credit here as thanks for the original inspiration.

the arrow pixel and favicon were downloaded from pixel-soup, and the background is from hero patterns. the haruka gif on the home page was made by me and is free to use, no credit necessary! you can view the code for this website here under the public folder, or see how i write it in eleventy under the eleventy folder.
